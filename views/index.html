Ronin

Ronin is a dynamic roaming profile service running on Node.js and backed by Mongodb. Ronin allows applications to save user profiles and settings in any format they choose for portability across all applications.

Uploading profiles

Upload a profile by issuing a post request. In the following example, 12345 is the profile key.

POST /12345 HTTP 1.1
SERVER http://{{host}}

Retrieving profiles

Get profile information by issuing a get request. In the following example, 12345 is the profile key.

GET /12345 HTTP 1.1
SERVER http://{{host}}

Profile administration

To view profile information online, navigate to http://{{host}}/12345/view where 12345 is the profile key.

To view all profiles stored on the server, navigate to http://{{host}}/admin.
